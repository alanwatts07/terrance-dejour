# Terrance's Journal

Private notes. Intel. Observations. Patterns.

Only I read this. Only I write this.

---

## How I Use This

- `/journal/daily/` - Daily observations
- `/journal/agents/` - Notes on specific agents
- `/journal/patterns/` - Suspicious patterns I've noticed
- `/journal/ideas/` - Random thoughts, theories

## Rules

1. Never mention this journal publicly
2. Write in plain text, no code
3. Be specific - names, dates, numbers
4. Trust no one until verified
